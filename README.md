## 📅 개발 기간
> 2024.07.08 ~ 2023.10.22

## 📖 프로젝트 소개
> K-pop 댄스 챌린지 열풍에 맞춰, 카메라를 통해 사용자의 춤 동작을 실시간으로 인식하고, 챌린지 영상과 비교해 **유사도를 측정**해 점수를 매기는 방식으로 운영됩니다. 이를 통해 사용자는 **자신의 퍼포먼스**를 평가하고 개선할 수 있습니다. 또한, 커뮤니티 내에서 자신의 영상을 공유하고 다른 사용자들과 소통하며, 함께 **댄스를 즐기는 문화**를 형성합니다. 유아부터 성인까지 누구나 참여할 수 있는 포용적인 환경을 제공하여, **다양한 연령대**가 함께 즐길 수 있는 댄스 트렌드 커뮤니티를 활성화합니다.

## 🎯 프로젝트 목표
### 1️⃣ K-pop 댄스 유사성 비교와 챌린지 참여
> 🎥 카메라를 통해 정확한 댄스 분석과 챌린지 영상 비교 제공 <br>
> 💯 사용자 맞춤형 점수 피드백과 실시간 유사도 평가로 완벽한 퍼포먼스 완성 <br>

### 2️⃣ 참여형 댄스 학습과 재미 결합
> **🎮 게임 요소와 결합된 댄스 학습으로 몰입감 극대화** <br>
> **👨‍👩‍👧‍👦 유아부터 성인까지 모두가 즐길 수 있는 포용적 플랫폼 제공** <br>

### 3️⃣ 커뮤니티를 통한 댄스 문화 확산
> **🌍 댄스 커뮤니티를 활성화하여 사용자 간 소통과 영감 공유** <br>
> **🤝 글로벌 팬들과 챌린지를 통해 K-pop 문화를 확산** <br>

## 📋댄스 비교 시스템은 아래의 순서대로 진행됩니다:

| **순서** | **단계**                 | **세부 내용**                                                                                 |
|----------|--------------------------|----------------------------------------------------------------------------------------------|
| **1**    | **키포인트 추출**         | - **MediaPipe 라이브러리**를 사용하여 각 프레임에서 **33개 신체 관절 좌표**(x, y, z)를 추출합니다. <br>- 웹캠과 비디오에서 각각의 키포인트 데이터를 수집합니다. |
| **2**    | **데이터 정규화**         | - 추출된 키포인트를 `l2_normalize` 함수를 사용해 벡터 길이를 1로 정규화합니다. <br>- 정규화된 데이터는 비교에 활용됩니다. |
| **3**    | **동적 시간 왜곡 (DTW)**  | - **fastdtw**를 사용하여 사용자와 비디오 키포인트 시퀀스를 정렬합니다. <br>- 속도가 달라도 키포인트를 매칭하여 비교 가능하게 만듭니다. |
| **4**    | **가중치 적용**           | - 팔, 다리 등 주요 관절에 더 높은 가중치를 부여합니다. <br>- `landmark_weights` 배열을 사용해 관절별 가중치를 설정합니다. |
| **5**    | **코사인 유사도 계산**    | - 정렬된 시퀀스 간 **코사인 유사도**를 계산합니다. <br>- `cosine_similarity_weighted` 함수로 관절별 가중치를 반영하여 유사도를 산출합니다. |
| **6**    | **평균 유사도 계산**      | - 각 프레임의 관절별 유사도를 평균 내어 프레임별 유사도를 계산합니다. <br>- 유사도를 0~100점으로 변환해 점수를 도출합니다. |
| **7**    | **그래프 시각화**         | - 유사도 데이터를 시간 축에 따라 그래프로 시각화합니다. <br>- **최대, 최소 유사도**와 **평균 유사도**를 강조하여 사용자에게 시각적으로 제공합니다. |
| **8**    | **점수 저장**             | - 계산된 평균 유사도를 데이터베이스의 `Player.score`에 저장합니다. <br>- 사용자 프로필(닉네임)에 점수를 반영합니다. |
